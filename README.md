# MBR's web portfolio

🖥️ What is this?

This is MBR's web portfolio. It is a website that showcases my work and presents me to whoever is watching the website.

🤖 AI usage

I'm using AI to learn and improve my HTML, CSS  as I did not have many before. I asked it how to do things. Some elements in the code are generated by AI.

📱 Features

The portfolio features an initial page thet allows you to navigate to the other two parts of the portfolio, the "About Me" page and the "Projects" page. It also has mobile support (Not perfect but it works,)

⚒️ How was it made?

The porfolio was made in VSCodium and I took advantage of its autocompletion tools, also AI assistance in coding with Gemini helped a lot!
